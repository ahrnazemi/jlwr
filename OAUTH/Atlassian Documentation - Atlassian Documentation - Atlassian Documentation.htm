          1. Rachel Robins [Atlassian Technical Writer]

            The sidebar remains visible when you view attachments, page history, page information from the Tools menu on the page. There is however a bug reported here CONF-28203 - In the Documentation Theme, it is impossible to toggle sidebar display when viewing Attachments Open  that the Show / Hide sidebar is not functioning correctly on the Attachments page.

            If you do want more screen space when viewing attachments, you can use the [ keyboard shortcut to hide the sidebar before you select Tools > Attachments. You'll need to return to the page view to show the sidebar again (because of the bug above).
